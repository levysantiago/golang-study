# Simple Web-server

This project is a simple web-server in Go.

## Versions

Go version: 1.21.4

## Run the project

```bash
go run main.go

```

Server will be running at `http://localhost:3000`

## Build project

```bash
go build

```

## Run built

```bash
./simple-webserver

```

Server will be running at `http://localhost:3000`
