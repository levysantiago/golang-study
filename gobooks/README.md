# Books CRUD

A CRUD of Books API with Golang and PostgreSQL.

## Versions

Go version: 1.21.4
Gorilla Mux: v1.8.1

## Start the database instance

```bash
docker compose up -d
```

## Run the project

```bash
go run main.go

```

Server will be running at `http://localhost:3333`

## Build project

```bash
go build

```

## Run built

```bash
./gomovies

```

Server will be running at `http://localhost:3333`
