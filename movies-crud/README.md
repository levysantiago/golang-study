# Movies CRUD

A CRUD of Movies API with Golang.

## Versions

Go version: 1.21.4
Gorilla Mux: v1.8.1

## Run the project

```bash
go run main.go

```

Server will be running at `http://localhost:3000`

## Build project

```bash
go build

```

## Run built

```bash
./movies-crud

```

Server will be running at `http://localhost:3000`
